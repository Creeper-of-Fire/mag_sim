# BBN 等离子体物理参数总结

## 1. 概述

BBN（Big Bang Nucleosynthesis）时期的宇宙是一个高温、高密度的等离子体。
温度范围约 **0.001 — 10 MeV**（$10^7$ — $10^{11}$ K），对应时间 **0.01 s — $10^6$ s**。

该等离子体包含以下粒子种类：

| 粒子 | 类型 | 存在温度范围 | 备注 |
|------|------|-------------|------|
| 光子 (γ) | 规范玻色子 | 全程 | 辐射主导 |
| 电子 (e⁻) | 轻子 | T > 0.5 MeV | 湮灭后数密度骤降 |
| 正电子 (e⁺) | 轻子 | T > 0.5 MeV | 与 e⁻ 几乎等量 |
| 电子中微子 (νₑ) | 轻子 | 全程 | 弱冻结后退耦 |
| μ/τ 中微子 (ν_μ, ν_τ) | 轻子 | 全程 | 已退耦 |
| 质子 (p) | 重子 | 全程 | 即 H-1 核 |
| 中子 (n) | 重子 | 全程 | 自由衰变，丰度下降 |
| 氘 (D) | 原子核 | T < 0.1 MeV | deuterium bottleneck 打开后 |
| 氦-3 (³He) | 原子核 | T < 0.1 MeV | |
| 氦-4 (⁴He) | 原子核 | T < 0.08 MeV | 主要产物 |
| 锂-7 (⁷Li) | 原子核 | T < 0.05 MeV | 微量 |

## 2. 等离子体特征参数

### 2.1 等离子体耦合参数 Γ

$$\Gamma = \frac{Z^2 e^2}{a_{WS} \, k_B T}$$

其中 $a_{WS} = (3/4\pi n_i)^{1/3}$ 是 Wigner-Seitz 半径。

| T (MeV) | Γ (e⁻) | 状态 |
|---------|--------|------|
| 8.6 | 0.012 | **弱耦合** (理想气体) |
| 0.7 | 0.012 | **弱耦合** |
| 0.08 | 0.005 | **弱耦合** |
| 0.03 | 0.0002 | **极弱耦合** |

**结论**: BBN 等离子体在整个温度范围内都是**弱耦合的**（Γ << 1），适合粒子模拟。

### 2.2 Debye 屏蔽长度 λ_D

$$\lambda_D = \sqrt{\frac{k_B T}{4\pi e^2 n_e}}$$

| T (MeV) | n_e (cm⁻³) | λ_D (cm) | c/H (cm) | λ_D / (c/H) |
|---------|------------|----------|----------|-------------|
| 8.6 | 9.6×10³⁴ | 7.0×10⁻¹² | 6.0×10⁸ | 1.2×10⁻²⁰ |
| 0.7 | 5.7×10³¹ | 8.4×10⁻¹¹ | 8.3×10¹⁰ | 1.0×10⁻²¹ |
| 0.08 | 7.3×10²⁷ | 2.5×10⁻⁹ | 1.0×10¹³ | 2.4×10⁻²² |

**结论**: λ_D 远小于 Hubble 半径（c/H），等离子体在局部热力学平衡的假设成立。

### 2.3 等离子体频率 ω_pe

$$\omega_{pe} = \sqrt{\frac{4\pi n_e e^2}{m_e}}$$

| T (MeV) | ω_pe (s⁻¹) | f_pe (Hz) | H (s⁻¹) | ω_pe / H |
|---------|-----------|-----------|---------|----------|
| 8.6 | 1.7×10²² | 2.8×10²¹ | 50 | 3.5×10²⁰ |
| 0.7 | 4.3×10²⁰ | 6.8×10¹⁹ | 0.36 | 1.2×10²¹ |
| 0.08 | 4.8×10¹⁸ | 7.7×10¹⁷ | 0.003 | 1.7×10²¹ |

**结论**: ω_pe >> H，等离子体振荡频率远快于宇宙膨胀，可忽略膨胀效应对局域等离子体动力学的影响。

### 2.4 粒子热速度

$$v_{th} = \sqrt{\frac{k_B T}{m}}$$

| T (MeV) | v_th(e⁻)/c | v_th(p)/c | 注 |
|---------|-----------|----------|-----|
| 8.6 | ≈ 1 (相对论) | 0.14 | 电子相对论 |
| 0.7 | ≈ 1 (相对论) | 0.039 | 电子相对论 |
| 0.08 | 0.55 | 0.013 | 电子半相对论 |
| 0.03 | 0.33 | 0.007 | 电子非相对论 |
| 0.003 | 0.11 | 0.002 | 非相对论 |

**结论**: 在 T > 0.5 MeV 时电子高度相对论，需要相对论 PIC 方法。离子（质子）在大部分温度范围内非相对论。

## 3. 离子（核子/原子核）的等离子体参数

### 3.1 离子等离子体频率 ω_pi

$$\omega_{pi} = \sqrt{\frac{4\pi n_i Z^2 e^2}{m_i}}$$

由于 $n_i \ll n_e$（重子-光子比 η ~ 6×10⁻¹⁰），$\omega_{pi} \ll \omega_{pe}$：

$$\frac{\omega_{pi}}{\omega_{pe}} \approx Z \sqrt{\frac{n_i m_e}{n_e m_p}} \sim Z \sqrt{\eta \frac{m_e}{m_p}} \sim 10^{-5} Z$$

### 3.2 离子声速 c_s

$$c_s = \sqrt{\frac{\gamma_e k_B T_e + \gamma_i k_B T_i}{m_i}}$$

在 BBN 时期 $T_e \approx T_i$：
- 单自由度 ($\gamma_i = 3$): $c_s \sim \sqrt{4 k_B T / m_p}$
- 对于 T = 0.1 MeV: $c_s \approx 0.03 c$

### 3.3 离子 Larmor 半径

在 BBN 时期没有宏观磁场（初始条件），但如果有磁不稳定性产生的原初磁场：
$$r_{Li} = \frac{m_i v_\perp}{Z e B}$$

## 4. PIC 模拟加入离子的关键考量

### 4.1 电荷中性

BBN 等离子体保持整体电中性：
$$n_{e^-} - n_{e^+} = \sum_i Z_i n_i$$

净电荷密度由重子数决定，远小于总电子数密度。

### 4.2 质量比问题

$m_p / m_e = 1836$。在 PIC 中精确模拟此质量比需要极高的时间分辨率（受电子等离子体频率限制），但可以采用：

1. **简化质量比**: 使用 $m_p/m_e = 100$ 或 400 以加速模拟
2. **隐式算法**: 跳过电子等离子体振荡的时间尺度
3. **混合方法**: 电子作为流体，离子作为粒子

### 4.3 核反应

BBN 核反应（如 p + n → D + γ）的截面远小于电磁散射截面。在 PIC 模拟中：
- 电磁相互作用自然由 PIC 算法处理
- 核反应需要额外的蒙特卡罗碰撞算子

### 4.4 建议的离子种类（按重要性排序）

| 优先级 | 粒子 | Z | A | 理由 |
|--------|------|---|---|------|
| 1 | 质子 (p) | 1 | 1 | 最丰富，唯一稳定的重子 |
| 2 | α 粒子 (⁴He) | 2 | 4 | BBN 主产物，质量分数 ~25% |
| 3 | 氘 (D) | 1 | 2 | 中间产物，对 D-bottleneck 敏感 |
| 4 | 电子 (e⁻) | -1 | — | 已有，需考虑离子耦合 |

## 5. 需要从 CosmoCons 获取的运行时参数

PIC 模拟在特定温度点运行时，需要从 CosmoCons 插值获取：

1. **数密度**: $n_e$, $n_p$, $n_\alpha$ 等 — 决定粒子权重和网格尺寸
2. **温度 T**: 决定粒子速度分布（Maxwell-Jüttner）
3. **Hubble 参数 H**: 决定膨胀速率，影响模拟盒大小的时间演化
4. **化学势 φ_e**: 影响电子-正电子平衡
5. **Debye 长度 λ_D**: 决定模拟网格分辨率要求
6. **等离子体频率 ω_pe**: 决定时间步长 $\Delta t < 2/\omega_{pe}$

## 6. 模拟参数估计

对于 T = 0.1 MeV 的典型 BBN 等离子体：

| 参数 | 值 | 备注 |
|------|-----|------|
| n_e | ~3×10²⁸ cm⁻³ | |
| λ_D | ~1.4×10⁻⁹ cm | 网格分辨率需 ≤ λ_D |
| ω_pe | ~1×10¹⁹ s⁻¹ | Δt < 6×10⁻²⁰ s |
| v_th(e) | ~0.5 c | 相对论 |
| v_th(p) | ~0.01 c | 非相对论 |
| n_p/n_e | ~6×10⁻¹⁰ | 重子占比极低 |

**注意**: 实际 PIC 模拟无法达到真实密度。通常使用缩放参数，保持关键无量纲数（Γ, ω_peΔt, λ_D/Δx）不变。
